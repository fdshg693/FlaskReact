JOINを使用してテーブルを結合する方法について説明します。

基本的なINNER JOINの構文：
SELECT table1.column, table2.column
FROM table1
INNER JOIN table2 ON table1.id = table2.foreign_id;
提案されたクエリ
SELECT u.name, o.order_date 
FROM users u 
INNER JOIN orders o ON u.id = o.user_id;
