# AI活用の課題

## 現状
Github Copilotは非常に便利だが、以下の点が課題として挙げられる。
### Chatでのやりとり
- ワークスペース全体のファイル構成や、現状使える全てのツールの情報が自動的にAIに渡される
    - そのため、ワークスペースが大規模になった際に、コンテキストが肥大化し、AIの応答品質が低下する可能性がある
    - カスタムエージェントの作成である程度は対応できるが、全てのケースをカバーするのは難しい（特に、ファイル数の肥大化には対策ができない）

### CLIでのやりとり
- ワークスペース全体の情報などは渡っていないようで、コンテキストは抑えられる。
    - ただ、まだまだ発展途上であり、細かいカスタマイズは、スクリプトなどで対応する必要がある

## 行いたいこと
- CODEXなどのCLIで動作する、カスタマイズ性が高いAIエージェントを活用して、コードレビューやドキュメント生成を自動化したい
- より厳密に、コミット単位・PR単位でAIによる自動化が行えるようにしたい
