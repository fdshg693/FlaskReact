# VSCODE推奨設定

```.vscode```フォルダ配下のsampleを書き換えて使用してください。
- launch.json
    - DEBUG構成
    - Streamlit用・他のpythonスクリプト実行用（FLASKもこれで立ち上げられる）
    - F5キーで実行可能
- settings.json
    - pythonのlintやformatの設定
    - PYTHONパスの設定
- extensions.json
    - 推奨拡張機能の設定
- tasks.json
    - 特定のタスクを実行するための設定
    - pytest, ruffフォーマット実行用タスクが設定されている
