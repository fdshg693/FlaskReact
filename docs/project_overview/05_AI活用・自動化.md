## 🤖 GitHub Copilot の活用

### 概要

本プロジェクトでは、GitHub Copilot の以下の機能を活用しています：

| 機能 | 説明 | 用途 |
|------|------|------|
| **エージェント** | 特定の役割を持つAIアシスタント | コードレビュー、ドキュメント生成 |
| **プロンプト** | 特定のタスクに特化したテンプレート | よく使う指示の再利用 |
| **タスク** | エージェントが実行するタスクの定義 | プロジェクト独自の仕組み |

### エージェント・プロンプト・タスクの関係

```
エージェント：役割とツールを定義
    ↑ 呼び出し
プロンプト：タスクをテンプレート化
    ↓ 参照
タスク：詳細な指示・ワークフロー
```

### 基本的な使い方

#### エージェントの呼び出し

1. GitHub Copilot Chat を開く（`Cmd+Shift+I`）
2. 画面上部のドロップダウンからエージェントを選択
3. 指示を入力

#### プロンプトの呼び出し

1. チャット画面で `/` を入力
2. 候補から選択
