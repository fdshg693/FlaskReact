# 開発の際に、使うと便利なスクリプト・コマンド集

# Github Copilot設定関係
- `python scripts/github_copilot/template_handle/create_default_prompt.py`
    - `.github_copilot_template`内に存在するエージェントに対応する、デフォルトのプロンプトファイルを生成
- `python scripts/github_copilot/template_handle/create_task_from_agent.py`
    - `.github/agents`配下に存在するエージェント定義をもとに、対応するタスクファイルを`.github/tasks`配下に生成
- `python scripts/github_copilot/template_handle/deploy_agents.py`
    - `.github_copilot_template`の内容をもとに、`.github/agents`配下にエージェント定義を生成・更新するスクリプト
    - 設定ファイルから、配置するエージェントを選択することも可能

# 品質担保系のスクリプト
- `python scripts/quality_check/example_checker.py`
    - プロジェクト配下の全ての.exampleファイルを取得し、それらに対する元のファイルが存在するかをチェックするスクリプト
    - .exampleファイルは全てGit管理下にあることを確認し、元のファイルはGit管理から外れていることを確認する
