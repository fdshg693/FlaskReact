# AI用の長期知識用ドキュメント
プロジェクトの全体像、開発規約等のcore knowledgeを格納します。  
一時的なタスクなどの情報は`memory`フォルダに格納してください。  

`.ai/knowledge/context`フォルダには、プロジェクトの全体像や目的、要件などのコンテキスト情報を格納します。  
`.ai/knowledge/entry`フォルダには、AIがプロジェクト概要をどのように把握すべきかを指示するエントリーポイント情報を格納します。
`.ai/knowledge/environment`フォルダには、開発環境や実行環境に関する知識を格納します。
`.ai/knowledge/guideline`フォルダには、コーディング規約やドキュメント作成規約などの開発ガイドラインに関する知識を格納します。
`.ai/knowledge/tools`フォルダには、AIが利用するツールに関する知識を格納します。
